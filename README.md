# Esther Couture — Portfolio Website

## Add your photos
Replace these placeholder files in the `images/` folder (keep the same filenames):
- `images/profile.jpg` — your photo, used in the hero section
- `images/look-1.jpg` through `images/look-8.jpg` — your outfit photos for the gallery
  (fewer than 8? just leave the rest out — placeholder tiles show automatically, nothing breaks)

## Update your real contact info
Open `index.html`, find the **CONTACT** section near the bottom, and replace:
- WhatsApp number and phone number
- Email address
- Instagram handle

## Host it on GitHub Pages
1. Create a new repository on [github.com](https://github.com) (e.g. `esther-couture`)
2. Upload these three files — `index.html`, `style.css`, `script.js` — plus the `images/` folder
3. Go to **Settings → Pages** in your repo, select the `main` branch, and save
4. Your live link will appear at `https://yourusername.github.io/esther-couture/`
5. Send that link (plus your GitHub repo link) to your lecturer's Gmail

That's it — good luck with the assignment! 🧵
